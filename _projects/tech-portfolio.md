## Project Overview

This portfolio website serves as my personal corner of the internet, showcasing my projects, skills, and professional journey.

## Features

### 🎨 Design
- **Light Theme**: Clean and modern aesthetic with a tech-inspired color palette
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle hover effects and transitions for better UX

### ⚡ Performance
- **Fast Loading**: Minimal dependencies, optimized assets
- **SEO Friendly**: Proper meta tags and semantic HTML
- **Accessible**: WCAG compliant design patterns

### 🛠 Technical Stack
- **Static Site Generator**: Jekyll
- **Styling**: Custom CSS with CSS Variables
- **Hosting**: GitHub Pages
- **Version Control**: Git & GitHub

## Key Learnings

1. **Jekyll Templating**: Learned how to create reusable layouts and includes
2. **CSS Architecture**: Organized styles using variables and logical sections
3. **Responsive Design**: Implemented mobile-first approach
4. **Git Workflow**: Managed version control and deployment
