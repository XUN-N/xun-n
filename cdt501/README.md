# CDT501 探索人工智能

**Course Code:** CDT501  
**Course Title:** Exploring Artificial Intelligence (探索人工智能)

## 课程简介

本课程系统介绍人工智能的基础理论与实践应用，涵盖机器学习、深度学习、自然语言处理、计算机视觉等核心领域。

## 课程目标

1. 理解 AI 的基本概念与发展历史
2. 掌握机器学习核心算法与实现
3. 了解深度学习框架与神经网络
4. 探索 NLP 与 CV 的实际应用

## 技术栈

- Python
- TensorFlow / PyTorch
- Scikit-learn
- OpenCV
- NLTK / spaCy
